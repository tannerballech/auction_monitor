Foreclosure Property Search Instructions

Please Note: This website provides information exclusively for foreclosure sales held in Tennessee. If you are seeking information for properties located in states other than Tennessee, you will not find sales information on this site.

Filtering Instructions:

Enter your filter criteria into one or more of the fields provided below.

File Number: This is an 8-digit reference number assigned by the firm, formatted as XX-XXXXXX. It uniquely identifies each file for easy reference.

Sale Scheduled Date: This field indicates the date when the foreclosure sale is scheduled to occur. Please enter the date in the format M/D/YYYY without leading zeros, such as 9/3/2024 or 12/3/2024.

Property Address: Enter the house number and street name for the property here. The search function recognizes standard abbreviations; however, please refrain from using a period (.) after the abbreviation.

Sale Crier: This refers to the individual responsible for conducting the foreclosure sale on the specified sale date.

After entering your information, press “Enter” or the “Tab” key to initiate the search.

The search results may take a few seconds to display.

Accessibility Assistance:

If you encounter any difficulties using this website, please call us at 470.321.7112 for assistance. Note that if you are not the mortgagor or property owner, RAS can only provide you with the information that is publicly available on this website.
